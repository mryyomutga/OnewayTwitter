# OnewayTwitter

最低限のTwitterの機能を備えたAndroidアプリ

## 機能

- ✔アプリ起動時にOAuth認証をして自分のTwitterアカウントと連携する
- ✔Tweet機能
- ✔自分のタイムラインを3件取得して表示
- 写真付きTweet
